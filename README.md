# ts-db-load-leb

- TypeScript 기반 **DB 적재 성능 측정 및 실시간 모니터링 서비스** 입니다.
- Worker(및 child process)를 사용하여 병렬로 데이터를 생성하고 다양한 데이터베이스에 적재하면서 지표를 수집하고 API를 통해 확인할 수 있다.

---

# 목적

- TypeScript 학습
- DB 연동 학습
- 실시간 모니터링 API 및 지표 제공

---

# 기술 스택

- Node.js
- TypeScript
- Express
- PostgreSQL
- MySQL
